NKO

2013-11-19


Node Knockout 是全球最大的Node.js编程比赛.NKO是为期48小时, 来自全球竞争者参与的在线虚拟Node.js黑客马拉松编程比赛.2013年比赛已于UTC 2013.11.9-2013.9.11举行, 比赛最终圣者由裁判和公共投票决定, 最终胜出的项目可到NKO网站查看. NKO举办灵感来源于Rails Rumble. NKO始于2010年, 今年是第四次举办.

Winners

NKO最终获胜作品已评选出来, 无论是对技术的运用, 设计, 实用性都比往届大大提高, 虽然离大规模产品化部署采用还有不少距离, 但已经有了一个很好得开始.

Rikoru (TEAM)

实时多人记忆游戏

Imdex (SOLO)

OCR-Indexed图片搜索引擎, 能搜索图片中的文本

Tidbit (INNOVATION)

可嵌入JS脚本, 能够通过网站用户的电脑挖矿为你赚钱

Zoom the Balloon (DESIGN)

有趣的游戏, 客户端应用

Video Funhouse (UTILITY/FUN)

完全运行于浏览器的视频转换工具

Harp Hook (COMPLETENESS)

基于Harp的Github page自动编译工具

Hub (POPULARITY)

Github + Chat = :D

其他

除了这些winner之外, 还有好多有趣, 好玩的作品比如基于npm的全新系统NodeOS, 感兴趣得同学可以到官网去逛一逛.

想参加明年比赛的同学可以订阅NKO的邮件通知,或follow NKO的twitter @node_knockout
